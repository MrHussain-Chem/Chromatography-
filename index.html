Core Definitions (MUST memorise exactly)

Chromatography: A technique used to separate and identify the components in a mixture.

Stationary phase: The phase that doesn't move (e.g., silica, alumina, paper)

Mobile phase: The phase that moves (e.g., solvent in TLC/CC, carrier gas in GC)

Adsorption: The process by which a solid holds molecules onto its surface

Separation Principles

Solid stationary phase (TLC/CC): Separation by adsorption
‚Üí Stronger adsorption = slower movement = lower Rf value
Liquid stationary phase (GC): Separation by relative solubility
‚Üí Greater solubility in stationary phase = slower movement = longer retention time

Factors Affecting Separation:

  • Polarity - polar compounds interact more with polar stationary phases
  • Charge - ion-exchange chromatography
  • Size - gel permeation chromatography
  • Stereoisomerism - chiral chromatography

Thin Layer Chromatography (TLC) - Practical Details

Method (6-mark answer structure):

  1. Dissolve a small sample in minimum solvent
  2. Draw pencil line 2cm from bottom (NOT pen - ink dissolves!)
  3. Spot sample on pencil line using capillary tube
  4. Place in beaker with 1cm depth solvent
  5. Seal with lid to saturate atmosphere
  6. Allow solvent to rise to near top
  7. Mark solvent front immediately
  8. Use UV light/ninhydrin/iodine to locate spots
Rf = distance moved by spot / distance moved by solvent
üìù Mark Scheme Alert: Always state Rf values to 2 decimal places (e.g., 0.45, not 0.5)

Column Chromatography (CC)

Key Method Points:

  • Fill column with stationary phase (silica/alumina)
  • Add filter/mineral wool plug at bottom
  • Dissolve sample in minimum solvent
  • Add to top of column
  • Open tap and add eluent continuously
  • Collect fractions - record retention time
Elution Order: Components that interact LEAST with stationary phase elute FIRST

Gas Chromatography (GC)

Key Features:

  • Very sensitive quantitative method
  • Tests wide range of compounds, especially low boiling point ones
  • Sample vaporises on injection
  • Components slow down as they interact with stationary phase
  • Exit at different times ‚Üí detected
üìä GC Chromatogram Analysis:
‚Ä¢ Number of peaks = number of components
‚Ä¢ Peak area ‚àù amount of component
‚Ä¢ Retention time identifies component (if reference available)

GC Limitations (common 2-3 mark question):

  • Many compounds have same retention time
  • High concentration peaks can hide smaller peaks
  • Unknowns have no reference
